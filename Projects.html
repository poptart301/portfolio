Scratch Game Project
Wild West Showdown

This is my first ever game I have created using code from Scratch. It is a two player game with one control for each player. I wanted to make a simple game but one that was fun to play. So I thought about a game two people can play together. The game is all about speed. Two cowboys must dual eachother. When the signal is shown they can quickdraw and fire at eachother once. The last one standing wins!

Python Text Adventure
Enter The Temple of Python

The Python Text Adventure project is a descriptive text adventure game where you will be placed in front of a ominous temple on a mission to obtain the hidden treasure. You must navigate through the dark temple and make difficult decisions to get to your destination. Make choices, find the treasure, and navigate your way back out of the temple safely!
